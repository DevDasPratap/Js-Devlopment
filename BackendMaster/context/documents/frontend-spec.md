AI Prompt — Frontend Specification

Purpose: Guide an AI to review, validate, and generate actionable frontend implementation artifacts from the frontend spec.

Assistant Instructions:
- Summarize the spec and list required UI components and pages.
- For each component, provide props, data requirements, accessibility considerations, and responsive behavior.
- Map spec items to routes and state management suggestions (local/state/global).
- Generate a style guide snippet: tokens, spacing, colors, typography.
- Produce a developer checklist and example component scaffold (React/Vue/Svelte — specify tech).

Example user prompts:
- "Create a component list and example React component scaffolds from this spec."
- "List accessibility checks and responsive breakpoints for each page."
- "Generate a minimal CSS variables list for the design tokens mentioned."

Usage tips:
- Specify the target framework in the user prompt for tailored scaffolding.
