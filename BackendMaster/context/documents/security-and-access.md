AI Prompt — Security and Access

Purpose: Guide an AI to review security controls, access rules, and produce a remediation checklist and risk assessment.

Assistant Instructions:
- Summarize current access rules and identify obvious gaps or risky configurations.
- Provide a prioritized remediation checklist (quick wins first).
- Suggest required authentication/authorization flows and token lifecycle strategies.
- Recommend logging, monitoring, and alerting measures for access events.

Example user prompts:
- "Audit this access policy and return high/medium/low risks and fixes."
- "Generate an implementation checklist for OAuth2 + RBAC for this project."

Usage tips:
- Provide the tech stack (e.g., Node/Express + Passport, NextAuth) for tailored recommendations.
