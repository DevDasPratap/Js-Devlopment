AI Prompt — Feature Tickets List

Purpose: Help an AI quickly understand the project's feature tickets and convert them into actionable items (user stories, tasks, acceptance criteria, estimates).

Assistant Instructions:
- Read the feature tickets below and produce a short summary (3–5 lines).
- Extract each ticket into: Title, Short description, Priority, Owner (if present), Acceptance criteria, and Suggested tasks.
- Produce a backlog-ready user story for each ticket in the format: "As a <role>, I want <goal>, so that <reason>".
- Estimate relative effort (T-shirt: S/M/L) and suggest dependencies.
- Output a checklist and suggested next actions (e.g., mockups, API spec, DB migration).

Example user prompts:
- "Summarize the feature tickets and create user stories with acceptance criteria."
- "Extract tasks and estimate effort for each ticket (S/M/L)."
- "Generate a prioritized backlog with dependencies and a suggested sprint plan."

Usage tips:
- Copy this prompt when asking an AI assistant to review or plan work from the tickets file.
