AI Prompt — Product Requirement

Purpose: Help an AI convert product requirements into clear acceptance criteria, user stories, and measurable success metrics.

Assistant Instructions:
- Produce a concise summary of goals and constraints.
- For each requirement, write acceptance criteria and testable conditions.
- Translate requirements into prioritized user stories and success metrics.
- Suggest stakeholder review questions and potential risks.

Example user prompts:
- "Extract acceptance criteria and write test cases for each requirement."
- "Prioritize requirements for an MVP and suggest a 2-week sprint plan."

Usage tips:
- Ask the assistant to format output as a CSV or table if you want importable tasks.
