AI Prompt — Technical Architecture

Purpose: Help an AI synthesize the technical architecture into diagrams, responsibilities, and implementation guidance.

Assistant Instructions:
- Produce a short architecture summary and list system components and responsibilities.
- Create a recommended deployment diagram and data flow description.
- Identify scalability and fault-tolerance suggestions (caching, queues, DB sharding, autoscaling).
- Produce an implementation roadmap with milestones (MVP, scale improvements, ops hardening).

Example user prompts:
- "Create a component diagram and short explanation for each service."
- "List trade-offs and recommended cloud services for hosting this architecture."

Usage tips:
- Ask for diagrams in Mermaid format for quick rendering in docs.
